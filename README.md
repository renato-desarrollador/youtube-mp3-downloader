# YouTube MP3 Downloader

Aplicación desarrollada en Python que permite descargar el audio de videos de YouTube en formato MP3 de forma sencilla desde la terminal.

Este proyecto fue creado como práctica personal para afianzar conocimientos de Python, manejo de librerías externas y control de versiones con Git y GitHub.

---

## Tecnologías utilizadas

- Python 3
- yt-dlp
- FFmpeg

---

## Funcionalidades

- Descarga audio desde una URL de YouTube
- Conversión automática a formato MP3
- Ejecución simple desde la terminal
